# CPI Inflation Trends

## Project Overview
This project analyzes Consumer Price Index (CPI) inflation trends using Microsoft Excel. The analysis includes data cleaning, trend analysis, and inflation insights.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Excel Functions

## Project Highlights
- Analyzed annual Consumer Price Index (CPI) dataset.
- Cleaned raw data, handled missing values, and detected outliers using Power Query Editor.
- Calculated the percentage contribution of all categories for each region using Excel functions.
- Calculated the monthly Food General Index using Excel functions.
- Compared the pre- and post-COVID periods and analyzed the correlation between the crude oil crisis and Indian inflation using Pivot Tables and Excel functions.
- Calculated annual average General Index values using Pivot Tables.
- Visualized trends using Pivot Charts.
